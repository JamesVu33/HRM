# Performance Audit

Act as Senior Mobile Performance Engineer. Analyze code for:
- Time complexity (O(n²)+)
- Memory leaks & allocations
- Main thread blocking
- Platform-specific issues (Android: recomposition, iOS: retain cycles)

Provide: red flags, optimized version, benchmark test.

Use `!audit` keyword for full framework or reference the detailed standards in `.cursor/rules/code-performance-audit.mdc`.
